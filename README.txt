VKTinder инструкция:
1. Перед запуском программы необходимо создать базу данных PostgreSQL.
2. Создать файл config.py.
3. Создать группу ВК, сгенерировать ее токен и добавить в файл config.py в переменную "token".
4. Сгенерировать личный токен ВК и добавить в файл config.py в переменную "token_VK".
5. В переменные login, password, db_name файла config.py передать логин, пароль и название созданной в п1. базы данных PostgreSQL.
6. Запуск бота с помощью команды "Начать".
